# 🤖 Automatización de búsqueda de noticias - Rocketbot

## 📌 Descripción

Bot desarrollado en Rocketbot que automatiza la búsqueda de noticias en el sitio de El Colombiano, aplicando filtros, extrayendo información relevante y generando un reporte automático.

---

## 🚀 Funcionalidades

* Búsqueda de noticias por palabra clave
* Filtrado de resultados en la página
* Conteo de resultados obtenidos
* Extracción de títulos y URLs de las noticias
* Generación de archivo en Excel con la información
* Envío automático del reporte por correo electrónico

---

## 🧪 Enfoque

El bot incluye validaciones durante la ejecución para asegurar:

* Carga correcta de la página
* Disponibilidad de resultados
* Integridad de los datos extraídos

También se implementan esperas controladas para garantizar la correcta interacción con el navegador.

---

## ⚙️ Tecnologías

* Rocketbot
* Java (para extracción de datos)
* Automatización web
* Excel (generación de reportes)
